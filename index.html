1. Is javascript (js) array mutable? Define: array methods filter, indexOf, at, splice and slice. Show one interesting example showing how to use these methods respectively?

Javascript arrays are mutable, which means that you can modify their contents by adding, removing, or updating elements.

Creates a new array with all elements that pass the test implemented by the provided function.

const numbers = [1, 2, 3, 4, 5];
const evenNumbers = numbers.filter(num => num % 2 === 0);
console.log(evenNumbers);

Returns the index of the first occurrence of a specified element in an array, or -1 if it is not found.

const fruits = ['apple', 'banana', 'orange'];
const index = fruits.indexOf('banana');
console.log(index);

Returns the element at the specified index in an array.

const numbers = [1, 2, 3, 4, 5];
const thirdNumber = numbers.at(2);
console.log(thirdNumber);

Changes the contents of an array by removing or replacing existing elements and/or adding new elements.

const numbers = [1, 2, 3, 4, 5];
numbers.splice(2, 1, 'three');
console.log(numbers); // Output: [1, 2, 'three', 4, 5]

Returns a shallow copy of a portion of an array into a new array object.

const numbers = [1, 2, 3, 4, 5];
const subarray = numbers.slice(2, 4);
console.log(subarray);

13. What are AJAX and XMLHttpRequest (XHR) objects respectively? Identify 3 properties and 3 methods.

A technique used in web development to send and receive data from a server asynchronously, without requiring the page to be reloaded. The technique relies on the XMLHttpRequest (XHR) object, which is a built-in JavaScript object that allows developers to make HTTP requests to a server and receive responses.

A number of properties and methods that can be used to control its behavior and handle the response. Here are three properties and three methods of the XMLHttpRequest object:

readyState: Indicates the current state of the request. There are five possible values, ranging from 0 (uninitialized) to 4 (completed).

status: The HTTP status code of the response. This property is set after the request has completed.

responseText: The response text from the server, if any. This property is set after the request has completed and the response has been received.

open(): Opens a new HTTP request, specifying the HTTP method (e.g., GET or POST) and the URL to send the request to.

send(): Sends the HTTP request to the server. If the request is asynchronous, this method will return immediately, before the response is received.

setRequestHeader(): Sets a request header for the HTTP request. This method must be called after the open() method and before the send() method.

LESS and SASS offer features that are not available in CSS, such as variables, functions, and mixins. These features allow developers to write more modular, reusable code and make it easier to maintain and update styles across a large project.

20. Explain Media Queries , viewport and grid layout.

Media Queries is a CSS technique that allows you to apply different styles to a web page based on certain conditions, such as screen size, orientation, or device type. This technique is particularly useful for creating responsive designs that adapt to different screen sizes. Media Queries allow you to define rules for specific breakpoints, making it possible to create a layout that looks great on all devices.

The viewport is the area of a web page that is visible to the user. It can vary depending on the size and orientation of the device being used to view the page. In order to create a responsive design, it's important to ensure that the viewport is set correctly. This can be done using the viewport meta tag, which allows you to control the width, height, and initial scale of the viewport.

Grid Layout is a CSS module that allows you to create complex layouts using a grid system. With Grid Layout, you can define rows and columns and place elements in specific cells, giving you fine-grained control over the layout of your page. This makes it possible to create responsive designs that adapt to different screen sizes, without relying on fixed-width layouts or media queries. Grid Layout is particularly useful for creating complex designs that require precise alignment and positioning of elements.
